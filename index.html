História da Web

voltar

A Web foi criada em 1988 1989 por Tim Berners-Lee como uma forma de compartilhar documentos entre cientistas.

No início, as páginas eram simples e feitas apenas em HTML.

Com passar dos anos, surgiram novas tecnologias como CSS e JavaScript, que deixaram os sites mais bonitos e interativos.

Hoje, a Web é essencial no dia a dia, usada para comunicação, estudos, trabalho e entretenimento.

O que são Clients

voltar

Clients são computadores, celulares ou programas que utilizamos para acessar informações e serviços na internet.

Eles funcionam como “a parte do usuário”, enviando pedidos para um servidor, que é responsável por responder com os dados solicitados.

Um exemplo comum de client é o navegador, como o Chrome ou o Firefox, que pede páginas da Web e exibe o conteúdo para o usuário.

Sem os clients, não seria possível interagir de forma prática com a Web e suas aplicações.

O que são Servers

voltar

Servers são computadores ou programas responsáveis por fornecer informações e serviços aos clients.

Eles recebem as requisições enviadas pelos usuários e respondem com os dados corretos, como páginas da Web, arquivos ou conteúdos de aplicativos.

Um exemplo comum de servidor é aquele que armazena sites e os entrega quando acessamos pelo navegador.

Sem os servers, a comunicação e o funcionamento da internet não seriam possíveis.

Linguagens de Programação

voltar

Linguagens de programação são formas de comunicação usadas para criar softwares, sites e aplicativos.

Elas permitem que os programadores escrevam instruções que os computadores conseguem entender e executar.

Existem várias linguagens, como Javascript Python, Java e C++, cada uma com suas características e usos específicos.

Sem as linguagens de programação, não seria possível desenvolver a tecnologia que utilizamos todos os dias.

Inspetor de Elementos

voltar

O Inspetor de Elementos é uma ferramenta disponível nos navegadores que permite visualizar e editar o código de uma página da Web em tempo real.

Com ele, é possível analisar a estrutura HTML, os estilos CSS e até testar modificações sem alterar o site original.

É muito útil para estudar páginas, corrigir erros e aprender como os sites são construídos.

Estrutura Básica do HTML

voltar

A estrutura do HTML define como uma página da Web é organizada.

Ela começa com a tag HTML, seguida pelas seções head, que contém informações sobre a página, e body, que contém o conteúdo visível para o usuário.

Dentro dessas seções, usamos outras tags, como h1 a h6, p e img, para organizar textos, títulos, imagens e outros elementos.

Ter uma boa estrutura garante que a página funcione corretamente e seja fácil de entender.

Falando sobre Tags

voltar

As tags HTML ajudam a organizar e formatar o conteúdo das páginas.

A tag i deixa o texto em itálico.

As tags h1 até h6 definem títulos e subtítulos, sendo h1 o mais importante e h6 o menos.

A tag title define o título da página que aparece na aba do navegador.

A tag input é usada para campos de formulário, como caixas de texto e botões.

E a tag img serve para inserir imagens na página.

Atributos das Tags

voltar

Atributos em HTML fornecem informações extras sobre os elementos.

Os atributos gerais podem ser usados em quase todas as tags, como id, class e style, que ajudam a identificar, estilizar ou organizar os elementos.

Já os atributos específicos só funcionam em determinadas tags, como src para imagens, href para links e type para campos de formulário.

Eles permitem personalizar o comportamento e a aparência dos elementos de forma precisa.

Textos

voltar

As tags de textos em HTML são usadas para organizar e formatar o conteúdo escrito nas páginas.

Por exemplo, p cria parágrafos, strong deixa o texto em negrito, i deixa em itálico, u sublinha e mark destaca com cor de fundo.

Já blockquote serve para citações e heading (h1 até h6) define títulos e subtítulos.

Essas tags ajudam a tornar o conteúdo mais claro e fácil de ler.

Listas Ordenadas e Não Ordenadas

voltar

As tags de listas em HTML são usadas para organizar itens de forma ordenada ou não ordenada.

A tag ul cria listas não ordenadas, com marcadores, enquanto a tag ol cria listas ordenadas, numeradas.

A tag li define cada item da lista.

Essas tags ajudam a estruturar informações de maneira clara e organizada nas páginas.
