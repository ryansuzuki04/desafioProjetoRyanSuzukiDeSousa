# Desafio de Projeto HTML

Este projeto é uma página web que criei como parte do meu desafio de HTML. O objetivo foi organizar e apresentar conteúdos sobre a história da Web, comunicação entre clients e servers, linguagens de programação e fundamentos do HTML.

## Estrutura e Conteúdos

A página contém seções bem organizadas com títulos, listas, links internos e externos, e campos de input para interação do usuário. Os principais conteúdos abordados incluem:

- **História da Web**: evolução da Web desde a criação por Tim Berners-Lee até tecnologias atuais como CSS e JavaScript.
- **Clients e Servers**: explicação sobre o papel de cada um na comunicação web.
- **Linguagens de Programação**: visão geral de linguagens como JavaScript, Python, Java e C++.
- **Fundamentos de HTML**: introdução às tags, atributos, estrutura básica, textos, listas e links.
- **Ferramentas**: apresentação do Visual Studio Code e Inspetor de Elementos para estudo e desenvolvimento.
- **Interatividade**: campos de input para nome, idade e cor favorita do usuário.

## Recursos e Boas Práticas

- Uso de **tags semânticas** (`h1` a `h6`, `p`, `ul`, `ol`, `li`, `blockquote`) para organizar o conteúdo.
- **Links internos** para navegação rápida entre seções e **links externos** para referências confiáveis.
- **Destaque de texto** com `strong`, `i`, `u` e `mark`.
- **Imagens** e **citações** para tornar a página mais atrativa.
- Estrutura clara e organizada, aplicando **boas práticas de HTML semântico**.
